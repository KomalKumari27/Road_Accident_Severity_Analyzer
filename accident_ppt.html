Safety Recommendations

Actionable outputs for authorities

Generated recommendation mapping

From hotspot risk factors → recommended interventions:

  • Rainy Weather → Reduce speed limits during rain
  • Poor Lighting → Install additional street lights
  • Driver Inattention → Increase awareness campaigns
  • Wet Roads → Improve road maintenance
  • Young Drivers → Provide defensive driving training
Why these actions
  • Addresses visibility, friction, and human-factor risks
  • Targets scenarios where severity shifts upward
  • Supports policy planning and enforcement priorities

Conclusion

Responsible use & next steps

Limitations
  • Based on historical data (not real-time traffic)
  • No GPS-based hotspot mapping
  • Performance depends on data quality & completeness
  • Predictions support decisions; they don’t replace experts
Future scope
  • Streamlit dashboard for severity input → recommendations
  • Live traffic API + weather API integration
  • GPS hotspot visualization
  • Try stronger models (XGBoost/LightGBM) or deep learning
Final takeaway

A complete workflow from EDA → ML severity prediction → hotspot pattern mining → targeted safety recommendations.
