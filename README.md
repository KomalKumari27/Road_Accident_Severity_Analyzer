# Road Accident Severity and Safety Hotspot Analyzer

## Project Overview
This project predicts road accident severity using machine learning and identifies high-risk accident patterns to support road safety planning.

## Problem Statement
Road accidents are a major public safety issue. This project predicts accident severity and analyzes accident patterns to identify risk factors and recommend safety measures.

## Real-World Impact
- Supports traffic police and transport authorities
- Helps identify accident-prone conditions
- Assists in road safety planning

## Dataset
Road Accident Severity Dataset (Kaggle)

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Engineering
5. Machine Learning
6. Model Evaluation
7. Hotspot Analysis
8. Safety Recommendations

## Machine Learning Models
- Logistic Regression
- Decision Tree
- Random Forest (Best Model)

## Results
- Best Model: Random Forest
- Evaluation Metrics: Accuracy, Precision, Recall, F1-score

## Future Improvements
